# WebGL Globe

Interactive WebGL globe with toggleable background.

## Setup

```bash
pnpm install
```

## Development

```bash
pnpm dev
```

Opens at http://localhost:3001

## Build

```bash
pnpm build
pnpm start
```

## Controls

Use the dashboard in the top-right corner to toggle:

- **Globe** - Show/hide the 3D globe
- **Background** - Show/hide the animated background
